Overzichtsscherm
Tijdens deze week kregen we de opdracht om onze Low-fi prototypes aan te passen en eventueel te verbeteren. Hierbij heb ik enkele zaken van de verschillende treinborden veranderd. Mijn eerste prototype van een vertrekbord was te gedetailleerd voor een low-fidelity prototype. In mijn nieuwe low-fidelity prototype heb ik allereerst symbolen en iconen toegevoegd. Deze kunnen verduidelijking aan de reizigers bieden en zijn verstaanbaar voor alle mensen, ongeacht de taal die ze spreken. Daarnaast heb ik de schets van mijn overzichtsscherm minder gedetailleerd gemaakt aangezien dit dichter aanleunt bij het concept van een low-fi prototype. Ook heb ik kleuren toegevoegd, zo hebben bijvoorbeeld alle IC-treinen een groene kleur gekregen, alle S-treinen een gele kleur enzovoort. Tot slot heb ik de vorm van mijn schets aangepast, en heb ik op schaal gewerkt. Zo heb ik gekeken naar hoe groot dit type scherm in het echt is, en heb ik deze maten genomen om op schaal 1:10 te werken.
Perronscherm
Ook bij dit scherm heb ik gelijkaardige veranderingen doorgevoerd als bij het overzichtsscherm (logische ordening, kleuren logisch toegepast, …)
Wagonscherm
Tot slot heb ik bij het wagonscherm ook verschillende zaken veranderd. Zo heb ik bij mijn nieuwe schets weer minder getailleerd gewerkt (animatie van de trein simpeler getekend). Daarnaast heb ik ook op schaal gewerkt, rekening houdend met de grootte van een echt scherm in deze categorie. Voor de rest heb ik al de andere zaken laten staan zoals ze op mijn eerste schets stonden.
